Make Gmail Your Default Email Client

An interactive guide to setting up Gmail on Windows 10.

1

Adjust Windows Settings

The first step is to tell Windows to use your web browser for handling email links. This change happens in the Windows "Default apps" settings. Follow the simple path below to make the change.

  1. Press the **Windows Key**, type **"Default apps"**, and press Enter.
  2. In the Default apps window, find the **Email** section.
  3. Click on the current default app (it might be 'Mail' or 'Outlook').
  4. From the list that appears, select the web browser you use most often (e.g., Google Chrome, Mozilla Firefox, Microsoft Edge).
2

Configure Your Browser

Now that your browser is set to handle email links, you need to tell it to use Gmail specifically. The process is slightly different for each browser. Select your browser below to see the specific instructions.

Google Chrome Instructions

  1. Open Gmail in a new Chrome tab: mail.google.com
  2. In the address bar, look for the protocol handler icon (a double-diamond shape: ◆).
  3. Click the icon, select **"Allow"** when asked if `mail.google.com` can open email links, and then click **"Done"**.
  4. If you don't see the icon, go to chrome://settings/handlers, ensure the main toggle is ON, and remove `mail.google.com` if it's in the "not allowed" list. Then, reload Gmail.

Mozilla Firefox Instructions

  1. Click the menu button (☰) in the top-right corner and select **Settings**.
  2. In the **General** panel, scroll down to the **Applications** section.
  3. Find the **mailto** Content Type in the list.
  4. Click the dropdown menu in the **Action** column next to it and select **Use Gmail**.

Microsoft Edge Instructions

  1. Open Edge settings (the "..." icon) and go to **Cookies and site permissions**.
  2. Scroll down and click on **Protocol handlers**.
  3. Ensure the setting **"Allow sites to ask to become default handlers for protocols"** is turned ON.
  4. Open Gmail in a new tab: mail.google.com
  5. Click the protocol handler icon (◆) in the address bar, select **"Allow"**, and then **"Done"**.
✓

Setup Complete!

You've configured your system to use Gmail for email links. Below are a few final points to keep in mind, along with a link to test your new setup.

Important Considerations

  • **Web-based vs. Desktop App:** This method directs email links to the Gmail website in your browser, not a standalone desktop application.
  • **MAPI Compatibility:** Some older desktop programs use a different system (MAPI) to send emails, which may not work with this web-based method. This is generally rare for modern applications.
